Pregunta 5: Uso de las etiquetas <ol> y <ul> y sus diferencias

Las etiquetas <ol> (ordered list), lista ordenada y <ul> (unordered list), lista desordenada se usan para crear listas en HTML. La diferencia principal es que <ol> crea una lista ordenada sea con números o letras, en cambio que <ul> crea una lista desordenada con viñetas.

Ejemplos de la lista ordenada (<ol>):

        
            <ol>
                <li>Primer ejemplo</li>
                <li>Segundo ejemplo</li>
                <li>Tercer ejemplo</li>
            </ol>
  1. Primer ejemplo
  2. Segundo ejemplo
  3. Tercer ejemplo

Ejemplo de la lista desordenada (<ul>):

        
            <ul>
                <li>Ejemplo 1</li>
                <li>Ejemplo 2</li>
                <li>Ejemplo 3</li>
            </ul>
  • Ejemplo 1
  • Ejemplo 2
  • Ejemplo 3
