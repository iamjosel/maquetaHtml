Pregunta 4: ¿Qué ocurre cuando se pasa de <h1> a una etiqueta <h6>?

Las etiquetas de encabezado (<h1> a <h6>) son usadas para definir títulos y subtítulos en un documento HTML. Cuando se va de <h1> a <h6>, el tamaño del texto disminuye. Por ejemplo:


Este es un encabezado <h1>

Este es un encabezado <h2>

Este es un encabezado <h3>

Este es un encabezado <h4>

Este es un encabezado <h5>
Este es un encabezado <h6>
