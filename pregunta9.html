Pregunta 9: Código CSS para negrita sin usar <b> o <strong>

Para colocar en negrilla a un texto sin usar las etiquetas <b> o <strong>, se puede utilizar la propiedad CSS font-weight. Por ejemplo:

        
            <!-- HTML -->
            <p class="negrilla">Texto en negrilla</p>

            <!-- CSS -->
            .negrilla {
                font-weight: bold;
            }
