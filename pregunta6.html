Pregunta 6: Utilidad del atributo "class"

"class" en HTML se utiliza para asignar una o más clases a un elemento, permiten aplicar estilos CSS, asimismo realizar selecciones en JavaScript de manera más eficiente y fácil. Por ejemplo, se puede aplicar un estilo a todos los elementos que tengan una clase específica.

Ejemplo:

        
            <div class="contenedor">
                <p class="texto-ejemplo"> Texto de ejemplo para la etiqueta class.</p>
                <p class="texto-normal"> Texto normal de ejemplo para la etiqueta class.</p>
                <button class="btn btn-primary"> Clic aquí</button>
            </div>
