Pregunta 10: Sintaxis CSS para cambiar fuente de elementos <p>

Para cambiar la fuente o tipo de letra de los elementos <p> en una página HTML haciendo uso de CSS, se puede usarfont-family. Por ejemplo:

        
            <!-- CSS -->
            p {
                font-family: 'Arial', sans-serif;
            }
