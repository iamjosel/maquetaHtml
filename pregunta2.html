Pregunta 2: Explicación de la etiqueta para el título principal

La etiqueta para el título principal dentro del contenido de una página web es <h1>. Esa etiqueta se usa para definir el encabezado de nivel 1 de la página, y generalmente es el título más importante de la página.

La diferencia entre <h1> y la etiqueta de título principal del documento o página web, que es <title>, está en el uso y ubicación, por ejemplo:

  • <h1>: Se usa en el cuerpo del documento HTML para establecer el título principal visible de la página web.
  • <title>: Se usa dentro del <head> del documento HTML para definir el título que aparece en la pestaña del navegador. Ejemplo: Pregunta 2
